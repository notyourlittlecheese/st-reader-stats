# ST Reader Stats

SillyTavern 的本地阅读统计扩展，提供：

- 当前聊天统计
- 当前角色全部聊天统计
- 全部角色总览与年度热力图
- IndexedDB 增量索引
- 当前选中 Swipe 汉字数
- 全部 Swipe 去重生成量
- Reroll 次数、活跃天数和聊天明细

## 安装

### 方式一：扩展安装器

将本目录上传到一个 Git 仓库，在 SillyTavern 的“扩展 → 安装扩展”中粘贴仓库地址。

### 方式二：手动安装

把整个 `st-reader-stats` 文件夹复制到：

```text
SillyTavern/data/<你的用户>/extensions/st-reader-stats
```

刷新 SillyTavern。在魔棒菜单中点击“阅读统计”。

## 统计口径

- **选中内容**：每条消息只统计当前 `swipe_id` 指向的内容。
- **Swipe 去重生成**：只统计助手消息的所有 Swipe；完全相同的 Swipe 记零新增，连续重复片段不重复计数。用户发送的文字仍属于“选中内容”，但不属于模型生成量。
- **Reroll**：每条消息按 `swipes.length - 1` 计算。
- 默认忽略角色开场白。
- 默认移除 `<think>`、`<thinking>` 和 `<reasoning>` 标签中的内容。
- 角色/全局统计目前只包含单人角色聊天，不包含群聊。

重复片段使用重叠字符窗口识别，默认阈值为 12。阈值越小，去重越激进；可在扩展设置中调整。

## 隐私与缓存

聊天内容不会发送到外部服务。索引保存在当前浏览器的 IndexedDB 中。

缓存指纹由聊天文件大小、消息数和末条消息时间组成。只有指纹变化的聊天会重新读取和计算。
